Tarixchi AI - Interactive History Learning Dashboard (Uzbek Language)
Overview
Create a premium, dark-themed educational dashboard web application called "Tarixchi AI" for learning Uzbek history. The interface should feel like a modern, gamified learning platform with rich visual content, interactive timelines, and AI-assisted tutoring features. The design language blends dark academia aesthetics with futuristic UI elements.
Design System
Color Palette
Background: Deep navy/black gradient (bg-gradient-to-br from-slate-900 via-[#0a1628] to-[#0f172a])
Primary Surface: Dark translucent cards (bg-slate-800/40 or bg-[#0f172a]/80) with backdrop blur (backdrop-blur-md)
Borders: Subtle cyan/teal borders (border-cyan-500/20 or border-teal-500/30)
Accent Colors:
Cyan/Teal for primary actions and highlights (text-cyan-400, bg-cyan-500, shadow-cyan-500/20)
Amber/Gold for gamification elements (XP, streaks, achievements)
Emerald for progress indicators
Text:
Primary: White (text-white)
Secondary: Slate gray (text-slate-300, text-slate-400)
Muted: Slate-500 for timestamps and metadata
Typography
Font Family: Inter or system-ui for body text; use a slightly decorative serif for historical headings if available
Headings: Bold, white, tracking-wide
Body: Regular weight, slate-300, leading-relaxed
Labels/Tags: Uppercase, tracking-wider, text-xs, font-semibold
Spacing & Layout
Container: Max-width 1440px, centered, generous padding (p-6 or p-8)
Grid System: CSS Grid with auto-fit columns, gap-6 between cards
Border Radius: Large rounded corners for cards (rounded-2xl or rounded-3xl), small for buttons (rounded-lg)
Shadows: Soft colored glows for active elements (shadow-lg shadow-cyan-500/10)

Interactions & Animations
Hover Effects: Cards lift up (-translate-y-1), border glow intensifies, shadow expands

Page Transitions: Smooth fade-in for carousel slides
Loading States: Skeleton screens with shimmer effect for content loading
Progress Ring: Animated SVG stroke-dashoffset on load
Responsive Behavior
Desktop: 3-column layout as described
Tablet: Left column collapses to top bar, center and right stack vertically
Mobile: Single column, hamburger menu for navigation, horizontal scroll for timeline
Technical Requirements
Use Tailwind CSS for all styling
Implement dark mode by default (no toggle needed)
Use CSS Grid and Flexbox for layouts
Ensure high contrast ratios for accessibility
Support RTL considerations for Uzbek text (though this is LTR)
Use CSS variables for theme colors to allow easy customization

Mood & Feel
The interface should feel like a premium educational game - engaging, rewarding, and intellectually stimulating. Dark mode creates focus, while cyan/teal accents provide energy and modernity. Historical content should feel respected and elevated, not dry or boring. The AI tutor adds personality and approachability to serious historical content.
