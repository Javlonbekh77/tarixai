Implement Wikimedia Commons image search provider.

Use official MediaWiki API.

Endpoint:
https://commons.wikimedia.org/w/api.php

Use action=query with generator=search or action=query list=search.
Search only files/images.

Return:
- image URL
- thumbnail URL if available
- title
- source page URL
- license metadata if available
- attribution if available

- Avoid SVG icons unless visualType is infographic/map and it is relevant.
- Filter out tiny images.
- Filter out irrelevant logos.

Return:

- attribution

====================================================
RANKING LOGIC
====================================================

Create rankImages(results, input).

Scoring:
+30 if title/source contains exact historical person/topic
+20 if provider is Wikimedia for historical-specific visual
+15 if provider is Openverse and license exists
+10 if landscape aspect ratio
+10 if image width >= 800
+10 if visualType matches title/tags
-30 if image appears logo/icon when visualType is not infographic
-20 if title is irrelevant
-15 if image is portrait when visualType is map
-15 if generic stock image for exact historical person
-50 if no imageUrl

For historical-specific scene:
Prefer:
1. Wikimedia
2. Openverse
3. Pexels/Unsplash only if no historical image found

For generic scene:
Pexels/Unsplash can be acceptable.

Return selectedImage and alternatives.

  "alternatives": [...]
}
